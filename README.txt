
// Instalación
//--------------------------------------------------------

1. Se tiene que bajar la ultima versión de nodejs. A continuación os dejo el link
https://nodejs.org/en/

2. Una vez instalado, abrimos la consola y vamos a la ruta donde se encuentra esta carpeta

	ejemplo ...  cd /ruta_de_la_carpeta

3. Una vez tenemos la ruta, instalamos los npm de dependencia que necesitamos. Escirmbimos en la consola:

	npm install

4.Instalar mongoDB 

	https://www.youtube.com/watch?v=1uFY60CESlM (Para Windows)
	https://docs.mongodb.org/manual/tutorial/install-mongodb-on-os-x/ (Para mac)

5. Arrancamos el programa una vez instalados los paquetes. Escribimos en la consola

	node app.js

6.El programa se incia en el puerto 3000, para ello abrimos el navegador y vamos a 

	localhost:3000

I esto es todo :)
